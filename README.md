# 甄嬛传十级学者 - 答题闯关游戏

一个基于《甄嬛传》的H5答题闯关小游戏。

## 游戏规则

- 从题库中随机抽取20道题目
- 每道题5分，满分100分
- 根据最终得分获得不同等级：
  - 0-40分：答应 🌸
  - 45-55分：常在 🌺
  - 60-70分：贵人 🌷
  - 75-85分：嫔 🌹
  - 90-95分：妃 👑
  - 100分：皇后 👑

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始答题"按钮
3. 选择答案后点击"下一题"
4. 完成20道题后查看结果

## 文件说明

- `index.html` - 主页面结构
- `style.css` - 样式文件（甄嬛传风格）
- `script.js` - 游戏逻辑
- `data.js` - 题库数据（100+道题目）

## 技术特点

- 纯前端实现，无需服务器
- 响应式设计，支持移动端
- 甄嬛传风格的UI设计
- 随机抽题算法
- 实时进度和得分显示

## 浏览器支持

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
